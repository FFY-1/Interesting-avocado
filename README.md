# English：
# Content
## 1 Items
- stay record
- Nevada record
- Avocado (edible)
- Roasted chicken leg (epic level) 10 hunger points, 2.3 nutrition value

## 2 Blocks
- FoodBlock: standing on it increases hunger and saturation
- noFoodBlock: standing on it increases hunger and saturation (opposite effect)

## 3 Dimension
- The only dimension is food: it's recommended not to go into the sea, no bedrock. Villages generate normally, there are Nether fortresses, and super gigantic trees made of FoodBlocks; the ground is mossy.

## 4 Easter Eggs
- In the overworld, FoodTrees events can generate some fun structures like Pigman fortresses; if lucky, there are also Nether fortresses and End cities, including an End ship!

## 5 Potions
- Spoiled potion (loses 2.5 health per second) causes nausea
